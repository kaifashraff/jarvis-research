## 🚀 QUICK START

### 1. View Lead Database
```bash
cd data
cat leads_database.json | python -m json.tool | head -50
```

### 2. Run Colab Notebook
1. Go to [Google Colab](https://colab.research.google.com)
2. Upload `colab/Zari_Business_Analysis.ipynb`
3. Enable GPU runtime
4. Run all cells

### 3. Start Automation
```bash
cd automation
python main_orchestrator.py
```

### 4. Launch Agent System
```bash
cd agents
python orchestrator.py
```
